#模拟电子点菜系统
##模拟餐厅点菜系统并设有简单的模拟VIP卡查询
###* 几个主要功能：
#####1.实现主推菜的自动转页和滑动展示
#####2.VIP卡预设的余额查询
#####3.菜单搜索功能
#####4.历史搜索功能
#####5.通过主界面中八个子图进入各类分菜单
